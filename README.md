# Gemini LLM Chatbot using Streamlit
# Project Overview

This project implements an interactive LLM Chatbot using Google Gemini API and Streamlit.

The chatbot allows users to interact with a Large Language Model in a conversational interface.

# Technologies Used

-Python

-Streamlit

-Google Gemini API

-python-dotenv


# Create a .env file and add:

GOOGLE_API_KEY=your_api_key_here

# Run the app
streamlit run app.py

# Features

-Conversational UI

-Session-based chat memory

-Powered by Gemini LLM

-Easy deployment
